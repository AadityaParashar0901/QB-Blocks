# QB Blocks 6
### QB Blocks - The Official QB64 based Minecraft Clone
*Made with OpenGL*

WARNING - Will not work if you have C++ Optimizations turned on  
Needs QB64PE v4.5.0+

## Terrain Generation
The terrain is generated in four phases:  
1. **Phase 1**: Generate Base Height
2. **Phase 2**: Fill Water
3. **Phase 3**: Generate Caves
4. **Phase 4**: Add Biome-Specific Features, Flora and Trees
